API Resources

Here you will see listed all the available resources along with their responses details.

Resources

Questionnare API supports the following resources:

  • Questions
  • Answers
  • Categories

Endpoints

For each resources you have available the following endpoints:

Question

  • GET api/v1/questions
  • GET api/v1/questions/<id>
  • POST api/v1/questions
  • PUT api/v1/questions/<id>
  • DELETE api/v1/questions/<id>

Answers

  • GET api/v1/answers/<id>
  • POST api/v1/answers
  • PUT api/v1/answers/<id>
  • DELETE api/v1/answers/<id>

Categories

  • GET api/v1/categories
  • GET api/v1/categories/<id>
  • POST api/v1/categories
  • PUT api/v1/categories/<id>
  • DELETE api/v1/categories/<id>
